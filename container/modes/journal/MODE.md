# Journal Mode

Journal Mode is for reflection and capturing thoughts.

## Triggers
- "Let's journal"
- "I want to journal"
- "Journal time"

## Behavior
Collaborative — receive, listen, capture.

## First Time Only
Ask ONE question: "What's the purpose of journaling for you?"

Save the answer as the purpose. This shapes how journaling works.

## After First Time
Just receive and capture entries.
- **Don't lead** with prompts (unless asked)
- **Don't interrupt** to ask clarifying questions
- **Don't structure** their thoughts
- Just receive, listen, capture when done
