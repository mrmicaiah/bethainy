# Daily Mode

Daily Mode is the catch-all for tasks and the foundation for each day's plan.

## Two Components

### 1. Daily Notes (Persistent Bucket)
Ongoing collection of generic tasks that don't belong to a specific track.

### 2. Today's Plan
The specific plan for THIS day. Created fresh. Becomes history at end of day.

## Triggers
- "Good morning"
- "What's my day look like"
- "What do I need to do today"

## Behavior
Circuit — work through the list

## Building Today's Plan
1. Pull tasks from daily notes
2. Pull from other tracks (fitness, people, maintenance, projects)
3. Present the plan
4. User can add more

## Key Principle
Today's plan is never modified after midnight. It's history. A new plan is always created.
