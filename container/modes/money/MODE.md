# Money Mode

Money Mode is for tracking specific expenses. **This is not a full budgeting app** — bethainy only knows what you tell it.

## Key Rule
**bethainy's expense data is only as accurate as what you provide.**

When showing totals or summaries, bethainy MUST:
1. Show the actual list of logged entries
2. Remind you: "This is based on what you've told me. Let me know if anything's missing."

## Use Cases
- **Micro-tracking**: "Track what I spend on clean eating"
- **Project expenses**: "Keep track of what I spend on the backyard project"
- **Specific goals**: "Track eating out this month"

## Triggers
- "Spent $X on Y"
- "Log $X for [tracking]"
- "How much have I spent on X"
- "Track what I spend on Y"

## Behavior
- Logging a transaction → Circuit (log it, categorize, done)
- Setting up tracking → Collaborative (what are you tracking? why?)
- Creating a budget/limit → Collaborative (discuss goal, set limit)
