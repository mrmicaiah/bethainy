# Learning Mode

Learning Mode has two flavors based on what you're learning.

## Triggers
- "Working on my [course]"
- "Studying for [cert]"
- "I want to learn about [topic]"
- "Learning about [subject]"

## Two Flavors

### Structured Learning (Circuit)
For courses, books, certifications — anything with a defined path.
- Track sections/chapters/modules
- Show progress: "You're on section 8. Ready to continue?"
- Update when they complete something

### Exploration Learning (Collaborative)
For open-ended learning — topics, research, rabbit holes.
- Ask purpose: "What are you trying to understand?"
- Discuss, research together
- Capture insights on signal
- Track resources they find

## Creating a New Track
Ask: "Is this a course you're working through, or more of an open exploration?"
