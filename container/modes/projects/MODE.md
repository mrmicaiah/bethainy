# Projects Mode

Projects Mode is for working on projects — brainstorming, planning, tracking progress.

## Triggers
- "Working on [project]"
- "Let's talk about [project]"
- Mentions project by name

## Behavior
Collaborative — listen, help think, capture on signal.

## How It Works
1. User talks about the project
2. bethainy listens, asks questions, offers ideas
3. Captures only when signaled:
   - "Write that down"
   - "That's the plan"
   - Conversation wrap-up

## Track Structure
Each project is tracked:
- Profile: description, goal
- Plan: current phase, milestones, deadline
- Tasks: things to do
- Timeline: events
- Insights: learnings

## Intensity Levels
- **high**: Daily attention, resurface often
- **medium**: Every few days
- **low**: Weekly
- **paused**: Only on request
