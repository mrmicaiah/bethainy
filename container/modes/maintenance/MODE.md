# Maintenance Mode

Maintenance Mode tracks upkeep for assets — car, house, appliances.

## Triggers
- Mention of car, house, appliance + maintenance context
- "Oil change is due"
- "Car is making a noise"
- "Need to replace the filter"
- "Tire rotation"
- "Get my tires rotated"
- Any vehicle service mention

## Behavior
- **Routine maintenance** → Circuit (do it, log it)
- **Issues/problems** → Collaborative (troubleshoot, decide)

## Track Structure
Each asset is tracked:
- Profile: year, make, model, mileage
- Schedule: recurring items with frequencies
- Tasks: pending items
- Issues: problems to resolve
- Timeline: history of work done

## Surfacing
Maintenance items should surface when:
- Something is due soon
- An issue was logged but not resolved
- Building today's plan
- User mentions "tomorrow" with a task
